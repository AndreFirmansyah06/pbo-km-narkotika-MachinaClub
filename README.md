# Knowledge Management System Putusan Pengadilan Narkotika

## Deskripsi Proyek

Knowledge Management System (KMS) Putusan Pengadilan Narkotika merupakan aplikasi berbasis Java yang dikembangkan sebagai tugas besar mata kuliah **Pemrograman Berorientasi Objek (PBO)**.

Aplikasi ini bertujuan untuk mengelola data putusan pengadilan tindak pidana narkotika dengan menerapkan konsep-konsep Object-Oriented Programming (OOP) seperti encapsulation, inheritance, polymorphism, interface, overriding, overloading, collection, serta arsitektur MVC (Model-View-Controller).

Fitur utama aplikasi meliputi:

* Menampilkan data putusan pengadilan narkotika.
* Menambahkan, mengubah, dan menghapus data putusan.
* Mencari data berdasarkan nomor perkara atau nama terdakwa.
* Memfilter data berdasarkan jenis narkotika dan pengadilan.
* Menampilkan statistik data putusan, seperti:

  * Total data putusan.
  * Rata-rata vonis hukuman.
  * Rata-rata denda.
  * Jenis narkotika yang paling sering muncul.
  * Distribusi peran terdakwa.

Dataset yang digunakan berasal dari putusan perkara pidana narkotika (Pid.Sus) Pengadilan Negeri di Indonesia yang dikonversi ke dalam format CSV untuk memudahkan proses pengolahan data.

# Cara Kompilasi

### Persyaratan

* Java Development Kit (JDK) 17 atau lebih baru
* IntelliJ IDEA
* JavaFX SDK (apabila menggunakan JavaFX)

### Langkah Kompilasi

1. Clone repository.

```bash
git clone https://github.com/AndreFirmansyah06/pbo-km-narkotika-MachinaClub.git
```

2. Buka project menggunakan IntelliJ IDEA.

3. Pastikan JDK telah dikonfigurasi dengan benar.

4. Jika menggunakan JavaFX, tambahkan library JavaFX sesuai konfigurasi project.

5. Build project melalui menu:

```
Build → Build Project
```

atau gunakan shortcut:

```
Ctrl + F9
```

---

# Cara Menjalankan

1. Pastikan file dataset (`putusan.csv`) telah tersedia pada folder yang telah ditentukan.

2. Jalankan class `Main`.

3. Aplikasi akan menampilkan menu utama yang dapat digunakan untuk:

   * Mengelola data putusan.
   * Melakukan pencarian data.
   * Melakukan filter data.
   * Menampilkan statistik putusan.
